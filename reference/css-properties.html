1. CSS Selectors How to target HTML elements

Tag / Element Selector

Targets ALL elements of that type.

p {
    color: #333;
}

/* Targets every <p> element on the page */

Class Selector .className

Targets elements with a specific class attribute. Most commonly used.

.header {
    background-color: #f4f4f4;
}

/* Targets: <header class="header"> */

ID Selector #idName

Targets ONE specific element. IDs must be unique per page.

#main-title {
    font-size: 2rem;
}

/* Targets: <h1 id="main-title"> */

Universal Selector *

Targets EVERY element. Used for resets.

* {
    margin: 0;
    padding: 0;
    font-family: Arial, sans-serif;
}

Pseudo-class Selector :hover

Targets an element in a specific state (hover, focus, first-child, etc.).

.logout:hover {
    background-color: #333;
    color: #fff;
}

/* Applied ONLY when the mouse is over .logout */
Live Demo — hover the button
Hover Me

Descendant Selector A B

Targets B elements that are inside A (any depth).

.navbar a {
    color: #fff;
    text-decoration: none;
}

/* Targets all <a> tags anywhere inside .navbar */

Child Selector A > B

Targets B elements that are direct children of A only.

.navbar > ul {
    list-style: none;
}

/* Targets <ul> that is a DIRECT child of .navbar */

Every HTML element is a rectangular box with four layers:

Space OUTSIDE the element (pushes other elements away).

/* All sides */

Space INSIDE the element (between border and content).

/* All sides */

/* Individual sides */

box-sizing

Controls how width/height is calculated.

/* Default: width = content only (padding + border add extra) */
box-sizing: content-box;

/* Recommended: width includes padding + border */
box-sizing: border-box;
Using box-sizing: border-box makes sizing predictable. A 200px-wide box stays 200px even with padding.

3. Display Property How elements behave

display

Controls how an element is rendered in the layout.

display: block;        /* Full width, stacks vertically (div, p, h1) */
display: inline;       /* Flows with text, no width/height (span, a) */
display: inline-block; /* Inline but respects width/height */
display: flex;         /* Flexbox container */
display: grid;         /* Grid container */
display: none;         /* Hides element completely */
display: block vs inline vs inline-block

block — takes full width

5. Positioning Where elements are placed

position

Controls how an element is positioned in the document flow.

position: static;   /* Default — normal flow */
position: relative; /* Offset from normal position */
position: absolute; /* Removed from flow, positioned relative to nearest positioned ancestor */
position: fixed;    /* Removed from flow, positioned relative to viewport */
position: sticky;   /* Normal flow until scroll threshold, then fixed */

position: sticky + top + z-index

Used for the header in Learning Logs — stays visible while scrolling.

.header {
    position: sticky;
    top: 0;        /* Stick at the top of the viewport */
    z-index: 1000; /* Stay above other elements */
}
z-index controls stacking order. Higher number = closer to the viewer. Only works on positioned elements (not static).

6. Typography & Text Controlling text appearance

font-family

Sets the typeface. Use fallback fonts separated by commas.

font-family: Arial, sans-serif;

font-size

Sets text size.

font-size: 16px;   /* Pixels — fixed size */
font-size: 1rem;   /* Relative to root element */
font-size: 1.2em;  /* Relative to parent element */

font-weight

Controls text thickness.

font-weight: 100;    /* Thin */
font-weight: 400;    /* Normal (same as "normal") */
font-weight: 500;    /* Medium */
font-weight: 700;    /* Bold (same as "bold") */

Horizontal alignment of text within a block element.

text-decoration

Controls underlines, overlines, and strikethroughs.

line-height

Controls spacing between lines of text.

list-style

Controls the bullet/number style on lists.

    list-style: none; /* Remove bullet points */
}

9. Sizing & Spacing Width, height, and overflow

width / max-width / min-width

width: 100%;        /* Full width of parent */
max-width: 100%;    /* Never wider than parent (prevents horizontal scroll) */
min-width: 200px;   /* Never narrower than 200px */

height / min-height

height: 100vh;    /* 100% of viewport height */
min-height: 40px; /* At least 40px tall */
vh = viewport height. 100vh means the full browser window height. This is how Learning Logs makes the page fill the screen.

overflow

Controls what happens when content is larger than its container.

overflow: visible; /* Content spills out (default) */
overflow: hidden;  /* Content is clipped */
overflow: scroll;  /* Always show scrollbars */
overflow: auto;    /* Show scrollbars only when needed */

10. CSS Variables Define once, use everywhere

:root + var()

Define variables in :root (the HTML element), use them anywhere with var(--name).

/* Define variables */
:root {
    --color-navbar-bg: #222;
    --color-text-light: #fff;
    --color-button-border: #333;
}

/* Use variables */
.navbar {
    background-color: var(--color-navbar-bg);
    color: var(--color-text-light);
}

.logout {
    border: 1px solid var(--color-button-border);
}
Change a variable in :root and it updates everywhere automatically. No find-and-replace needed.

11. Responsive Design Adapting to screen sizes

@media queries

Apply CSS rules only when certain conditions are met (usually screen width).

/* Only applies when browser width is 700px or less */
@media (max-width: 700px) {
    .logo h3 {
        display: none; /* Hide "Learning Log" text on mobile */
    }
}
Resize your browser window to see this in action
📖 Learning Log ( ← this text hides below 700px )

Common breakpoints

@media (max-width: 700px)  { /* Tablets and below */ }
@media (max-width: 480px)  { /* Mobile phones */ }
@media (min-width: 1024px) { /* Desktops and above */ }

Viewport meta tag

Required in HTML <head> for responsive design to work on mobile devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0" />
Without this tag, mobile browsers zoom out and show the desktop version of the page.
